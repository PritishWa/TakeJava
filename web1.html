*Font-color*
  • What is Font-color ?

    • Color is sometimes used to convey meaning beyond the basic text. for example, you may use color to emphasize an important statement.Using color to communicate meaning or emphasis is problematic for students with color blindness. If you want to use color to express meaning, you should also provide a supplemental means to convey the information without color.

  • Syntax

    • The color is an attribute of font tag. which Specifies The Text color

    font color = "color_name[rgb_number or hex_number"]
  1. Color name:

    2. font color= "Blue"

  2. Rgb_Number:

    3. font color= "rgb(128,128,0)"

  3. Hex_Number:

    4. font color= "#00FF00"
